commandes git pour tout faire depuis le début:
aller dans le répertoire du projet
>git init
dans le .gitignore ajouter les trucs a ignorer (*.pyc, *.pyo, __pycache__, ...)
>git add *
git status permet de verifier ou on en est
>git commit -m "mon message"
informer git qu'un depot existe sur github
>git remote add origin https://github.com/nomutilisateur/MonProjet.git
(Nous avons ajouté après le mot origin pour indiquer un nouvel endroit à partir duquel viendront les fichiers. remote est un descripteur de origin, pour indiquer que l’original n’est pas sur l’ordinateur, mais quelque part en ligne.)
pour lister toutes les origines distantes:
>git remote -v
on envoie tout:
>git push --set-upstream origin master
Si ça bloque: git push -f origin
